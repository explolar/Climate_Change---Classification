## üß† Methodology Summary

### ‚úÖ Standardized Precipitation Index (SPI)
- Gamma distribution fitting
- Rolling sums (SPI-3, SPI-6, SPI-12)
- Z-score normalization
- Supports **two base periods** (e.g., 1975-1997 & 1998-2020)

### ‚úÖ Statistical Tests
- Mann-Kendall for trend detection
- Wilcoxon-Mann-Whitney for median shift
- K-S test for distribution fitting
- Randomness validation for stochastic properties
